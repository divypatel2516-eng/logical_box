# logical_box
# Pattern Generator and Number Analyzer

## Description
This is a Python menu-driven program that allows users to:
1. Generate a star pattern.
2. Analyze a range of numbers.
3. Exit the program.

## Features
- Generate triangle star patterns.
- Check whether numbers are Even or Odd.
- Calculate the sum of numbers in a given range.
- User-friendly menu interface.

## Requirements
- Python 3.x

## How to Run

1. Open terminal.
2. Navigate to the project folder.
3. Run:
## log!cbox.py
